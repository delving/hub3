IndexService.GetIndexes

GetIndexes gets a list of Indexes.

JSON JavaScript iOS Go

JSON input

{}

Empty object {}

JSON output

{
	"indexes": [
		{
			"indexPath": "firesearch/indexes/my-index",
			"name": "All Products Index",
			"language": "spanish",
			"keepStopWords": false,
			"caseSensitive": false,
			"noStem": false
		}
	],
	"error": "something went wrong"
}
.indexes (array of Index) Indexes are the indexes managed by this service.
.indexes[i].indexPath (string) IndexPath is the collection path in Firestore for this index. Each index must use a unique path.
.indexes[i].name (string) Name is an internal human readable name for this index. End users will never see this.
.indexes[i].language (string) Language of the index.
Only: ["english","dutch","french","german","russian","spanish","swedish"]
Default: "english"
.indexes[i].keepStopWords (bool) KeepStopWords prevents stop words from being removed from this index.
Default: false
CAUTION: Will result in larger indexes.
.indexes[i].caseSensitive (bool) CaseSensitive preserves case across this index. By default, all entries and queries are lowercased.
Default: false
.indexes[i].noStem (bool) NoStem prevents words from being reduced. Only effective if a Language is specified.
Default: false
CAUTION: Will result in larger indexes.
.error (string) Error is string explaining what went wrong. Empty if everything was fine.
